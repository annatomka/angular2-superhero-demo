Welcome

This is a simple voting app created with AngularJS 2.0.0-alpha.45, Firebase and Materialize CSS.

Demo

You can see the application in action in this Demo.

Installation

Install dependencies:

$ npm install

Build project: 

$ gulp

How to use

Compile code and watch for changes: 

$ npm run tsc

Start local http server run the following command in the root directory of the application: 

$ npm start
